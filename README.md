<a name="english"></a>
## :gb: English

### Overview

A Flask application for converting multimedia files with FFmpeg and generating audio from text. Supports various video and audio formats.

### Features

- **Video Conversion** - Convert between video formats
- **Audio Extraction** - Extract audio from video
- **Text-to-Speech** - Generate audio from text
- **Format Support** - MP4, AVI, MKV, MP3, WAV
- **Web Interface** - Easy-to-use Flask UI

### Quick Start

```bash
git clone https://github.com/fracabu/ffmpeg_converter_project.git
cd ffmpeg_converter_project
pip install -r requirements.txt
python app.py
```

Access at `http://localhost:5000`
